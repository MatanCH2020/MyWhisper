מה קורה אחרי שמריצים?

  1. הכנה אוטומטית

    אם Git חסר במחשב — הוא מותקן לבד. אין מה לעשות, רק להמתין.

  2. הפרויקט יורד

    הקבצים נשמרים אצלך תחת %USERPROFILE%\MyWhisper.

  3. המנוע מותקן

    Python וכל הספריות (כולל CUDA לכרטיס NVIDIA) מותקנים בסביבה מבודדת. זה השלב הארוך — כמה דקות, תלוי במהירות האינטרנט.

  4. קיצור דרך נוצר

    אייקון MyWhisper מופיע על שולחן העבודה — כך פותחים את התוכנה בפעם הבאה.

  5. האפליקציה עולה

    החלון נפתח, והתוכנה יושבת במגש המערכת (ליד השעון) ומחכה לקיצור.

  6. הרצה ראשונה — הורדת המודל

    מודל התמלול יורד פעם אחת (~1.5–3GB). האייקון במגש כחול בזמן הטעינה והופך אפור כשמוכן. מכאן — Ctrl + Space, מדברים, וזה עובד.

שאלות נפוצות

כל מה שרצית לדעת

מה צריך כדי שזה ירוץ?

Windows 10/11 ומיקרופון. מומלץ כרטיס NVIDIA (מהיר מאוד); בלעדיו התוכנה עוברת אוטומטית למעבד — עובד, רק איטי יותר. ההתקנה מסדרת לבד את Python וכל התלויות.

כמה זה עולה?

כלום. קוד פתוח, רץ מקומית על החומרה שלך. אין מנוי ואין הגבלת שימוש.

המידע שלי נשלח לענן?

לא. המודל יורד פעם אחת (כ-1.5–3GB) ומאז הכול רץ אופליין. אף הקלטה או תמלול לא עוזבים את המחשב.

איך משנים את קיצור המקלדת?

ישירות מתוך הגדרות ← קיצור מקלדת — לוחצים ומקישים צירוף חדש (למשל Ctrl+Alt+Space), והוא מתעדכן מיד. יש גם כפתורי צירוף-מהיר.

Windows מציג אזהרה על קובץ ההתקנה?

זה SmartScreen על קבצים חדשים ללא מוניטין. הקוד פתוח וניתן לקריאה ב-GitHub; אפשר גם להתקין בפקודת ה-PowerShell שלמעלה.

בטעות התחלתי הקלטה / שכחתי מיקרופון פתוח?

Esc באמצע הקלטה מבטל בלי לתמלל, והקלטה שנשכחה נעצרת אוטומטית אחרי 10 דקות.

מוכן להפסיק להקליד?

התקנה בפקודה אחת. הקיצור פעיל תוך דקות.
